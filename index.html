CSS Box Model Cheatsheet

What is the CSS Box Model?

The CSS Box Model is a fundamental concept that describes how elements are structured in CSS. Every element is essentially a box, consisting of four main areas:

  • Content: The actual content of the element (text, image, etc.).
  • Padding: The space between the content and the border.
  • Border: The outer edge of the element that surrounds the padding and content.
  • Margin: The space outside the border that separates the element from others.

Box Model Properties

Property Description Example
margin Defines the space outside the border. You can set margin for top, right, bottom, and left sides separately. margin: 10px;
border Sets the border around the element. You can set border width, style, and color. border: 1px solid black;
padding Defines the space between the content and the border. padding: 10px;
height Specifies the height of the content area. height: 100px;
width Specifies the width of the content area. width: 200px;

Box Model Example

In the above example, the content area is inside the padding, border, and margin layers. The dimensions of the box are adjusted based on these properties.
